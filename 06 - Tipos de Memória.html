Hierarquia de memórias

  1. Registradores
  2. Cache
  3. Memória principal: RAM volátil
    ROM não volátil
  4. Memória secundária: HD, SSD, CD/DVD, pen drive
  1. quanto mais alto mais caro, menos armazenamento e mais rápido
  2. quanto mais baixo mais barato e mais armazenamento e mais lento

ROM não rápida e menor q o CPU MB e GB médio Secundária HD, SSD ... não baixa GB e TB baixo

Registradores

  • volátil
  • extremamente rápido
  • disponibilidade limitada
  • dentro do CPU
  • armazena dados temporarios para para o CPU processar

Cache

  • volátil
  • muito rápida
  • L1: Cache interna
  • L2: Cache interna
  • auxiliar da RAM por ser mais rápida
  • Função: acelerar a velocidade de transferencia das informações entre CPU e memória principal

Cache interna L1

sub-dividida em dois modulos
  1. L1d:
    16 - 128 KB
    dados
  2. L1i:
    16 - 64 KB
    instruções

Cache externa L2

SRAM Static Random Access Memory 2 - 8 MB

faz a comunicação com a memória principal DRAM Dynamic Random Access Memory

Memória Principal

  • dá acesso às memórias secundárias
  • aloca dados e instruções de programas
  • volátil
  • a memória RAM é denominada DRAM por precisar de refrescamento da memória
  • as DRAMs consomem muitos ciclos do processador para a realimentação

a frequencia do ciclo de clock é medido em Hertz (HZ)

diferenças entre DDR2, DDR3 ... está na velocidade, latência, voltagem ...
